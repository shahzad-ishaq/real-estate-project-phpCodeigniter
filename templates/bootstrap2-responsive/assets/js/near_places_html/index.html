Near Places

Near Places is a jQuery plugin that utilizes Google map components.

Getting Started

Dependencies

Requires jQuery v1.8.0+, Google map components here .

<!-- Jquery -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.1/jquery.js"></script>

<!-- Googlr map -->
<script src="http://maps.googleapis.com/maps/api/js?libraries=weather,geometry,visualization,places,drawing&&key={API_KEY}"></script>

<!-- Googlr map Infobox -->
<script src="js/map_infobox.js"></script>

<!-- Googlr map Markerclusterer -->
<script src="js/map_infobox.js"></script>

Install on page

Add js and css files from plugin:

<!-- Css -->
<link rel="stylesheet" href="css/near_places.css">

<!-- Js -->
<script src="js/near_places.js"></script>

Usage

Create your <select> with the .selectpicker class. The data-api will automatically theme these elements.

<div> id="property-map-plugin"></div>

Options can be passed via data attributes or JavaScript.

var map_near = $('#property-map-plugin');
    map_near.near_places();
    map_near.near_places('setMarker', {ltng :[45.812231, 15.920618], 'address':'Ilica 345, HR-10000'})
