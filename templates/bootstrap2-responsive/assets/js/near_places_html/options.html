Core options

Default config for near places

You can changed when near_places init

        
    map_near.near_places({
        'mapOptions': {
                'zoom' : 10,
            }
    });

Settings for change

        
    settings = {
        'places': {
            'health': {
                'icon': 'img/hospital.png',
                'type': 'hospital,health',
                'title': 'Health',
            },
        },
        'mapOptions': {
            'zoom': '15',
            'mapTypeId': 'ROADMAP',
            'scrollwheel': false,
            'styles': '',
            'radius': '2000',
            'default_place_id': '4',
        }
    }
